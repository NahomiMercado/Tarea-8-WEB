# Tarea 8 - Web Applications

This project was developed with Laravel as part of Activity 8.

## Description
The system allows users to register, log in, access a protected dashboard, and log out securely.  
It also sends an alert email after a successful login.

## Features
- User registration
- User login
- Protected dashboard
- User logout
- Login alert email
- Cloud email testing with Mailtrap
